## 개요

'한글 팡팡 게임'의 전체 구조(로그인·범위·환경설정·구글시트 연동·결과분석·음성·BGM·손동작·QR)를
**그대로 재사용**하고, **게임 방식만** 사자성어 4지선다로 교체한 신규 게임.

- 한글팡팡(`pangpang/`)과는 **완전히 별개**의 저장소·구글시트·localStorage를 사용한다.
- localStorage 접두어: `hpg_` → **`scj_`** (기록·설정 분리)
- git 원격: **`cjhong7/idioms`** (한글팡팡 `cjhong7/pangpang`과 분리, 이력 초기화)

## 게임 방식 (한글팡팡과 다른 핵심)

- 화면 **중앙에 사자성어의 '뜻'** 을 제시어로 보여줌
- **동서남북(N/E/S/W)** 에 ①②③④ 번호를 붙인 **4개 보기**(정답 1 + 오답 3) 표시
  - 보기 표시: **한글음 + 한자 병기** (예: 일석이조 / 一石二鳥)
- 알맞은 사자성어를 **터치(또는 손동작)** 로 선택
- **60초 / 20문제**, 정답을 맞히면 다음 문제로 진행 (정답 맞힐 때까지 같은 문제 유지)

### 단계 · 점수

| 단계 | 수준 | 정답 | 오답 |
|------|------|------|------|
| 저학년 | 초등 1~2학년 | **+4점** | **-2점** |
| 중학년 | 초등 3~4학년 | **+6점** | **-3점** |
| 고학년 | 초등 5학년 이상 | **+8점** | **-4점** |

- 점수는 0점 미만으로 내려가지 않음
- 오답 보기는 비활성화(어둡게)되고 감점, 정답을 고를 때까지 계속 시도
- '이어서 하기' = 저학년 → 중학년 → 고학년 순으로 진행

### 내장 사자성어 데이터 (166개)

| 단계 | 개수 |
|------|------|
| 저학년 | 52개 |
| 중학년 | 54개 |
| 고학년 | 60개 |

- 데이터 형식: `{i:'한글음', h:'한자', m:'뜻'}`
- 단계 내/단계 간 중복 없음, 한자 모두 4자 검증 완료
- `index.html`의 `const DEF = {1:[...], 2:[...], 3:[...]}` 에 정의 (1=저, 2=중, 3=고)

## 구현 기능 (한글팡팡에서 그대로 계승)

- 모드 선택(오프라인/온라인) · 온라인 로그인(학생참여/참여코드) · 데이터 범위 선택(전체/학년/반)
- 인트로(명예의 전당) · 음성 이름 입력(실패 시 키보드) · 카운트다운
- 손동작 인식(MediaPipe Hands) — 게임 중 4개 보기 위에 손 **0.5초 체류**로 선택, 메뉴는 0.7초 드웰
- 결과 분석(통계·필터·상세표·자주 틀리는 사자성어·성장 추이·엑셀 다운로드)
- 환경설정(관리자 비번 · 학교 설정 · 구글 드라이브 연동 · 참여코드/QR 생성 · BGM · 손동작 · 엑셀 등록)
- 효과(파티클·플래시·콤보·효과음) · 명예의 전당(범위별 분리, 최대 200개)
- 자주 틀린 사자성어 가중치 출제(적응형 큐)

### 엑셀로 사자성어 등록 (커스텀)

- 첫 행 머리글: **저학년 / 중학년 / 고학년**
- 각 칸 형식: **`한글음|한자|뜻`** (구분자 `|` `/` `,` 모두 가능)
  - 예: `일석이조|一石二鳥|한 번에 두 가지 이익을 얻음`

## 구글 드라이브 연동 (GAS)

- 한글팡팡과 **동일한 GAS 코드 구조**, 단 **새 스프레드시트**에 연결한다.
- 환경설정 → 구글 드라이브 연동 → `📖 설정 방법` → 📋 코드 복사 → Apps Script 붙여넣기 → 새 배포(웹앱)
- '단계' 열에는 1/2/3(저/중/고)이 저장되며, 화면 표시 시 STAGE_NAME으로 변환된다.
- ⚠️ 한글팡팡 시트가 아니라 **새로 만든 시트**의 GAS URL을 사용할 것.

## 구글 드라이브 연동 현황

### 설정 완료 후 데이터 확인 위치
- 스프레드시트 → **`전체점수` 탭** (기본 `시트1` 탭이 아님!)
- 학년별 저장 시 → `N학년점수` 탭 자동 생성
- 반별 저장 시 → `N-M점수` 탭 자동 생성

### 점수가 시트에 안 보일 때 체크리스트
1. ☑ 게임을 **온라인 모드**로 플레이했는가?
2. ☑ 환경설정 → GAS URL이 **입력**되어 있는가?
3. ☑ 게임 종료화면에 **`✅ 클라우드 저장 완료`** 가 떴는가?
4. ☑ 스프레드시트에서 **`전체점수` 탭**을 열었는가? (`시트1` ❌)
5. ☑ GAS 배포 시 액세스 권한을 **`모든 사용자`** 로 했는가?

### 종료화면 메시지 → 원인 대응표
| 메시지 | 원인 |
|--------|------|
| `✅ 클라우드 저장 완료` | 정상 — `전체점수` 탭 확인 |
| `❌ GAS 응답 오류 — 배포(액세스: 모든 사용자) 확인` | GAS 배포 권한이 "나만"으로 설정됨 |
| `❌ 저장 실패: 비밀번호 불일치` | 비밀번호 불일치 |
| `❌ 연결 실패` | GAS URL 오류 또는 네트워크 문제 |
| `⚠️ Google Sheets 미연동` | 환경설정에 GAS URL 미입력 |
